# UAF-Quantum – квантовая теория матрёшечной когерентности

**UAF-Q (Universal Accounting Field – Quantum)** – оригинальная квантовая рамка, в которой время возникает из роста корреляций (запутанности) между уровнями иерархии, а стрелка времени – из баланса между интеграцией и декогеренцией.

**Ключевое уравнение**  
`∂Γ_l/∂τ = η·(∑Φ·ℰ)·(1-α_l)² – λ·ΔS_env·α_l`  
где `Γ_l` – когерентность будущего на уровне `l`.

## Содержимое

- `README.md` – общее описание.

- `equations/` – LaTeX‑файлы со всеми системными уравнениями.
- `simulations/` – Python‑код для численных экспериментов (ODE, стохастика, графики).
- `docs/` – сравнение со стандартными подходами (Lindblad, Page‑Wootters).
